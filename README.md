 
# MVP Aula-USM UI  

Este proyecto corresponde al frontend de **Aula USM**, desarrollado con **React + Vite**.

## Levantamiento

Para ejecutar o compilar la aplicación correctamente, asegúrate de estar ubicado en la carpeta:

```
mvp_aula_usm/aula-ui
```
 
### 1. Instalar dependencias

Al descargar o clonar el proyecto, primero debes instalar todas las dependencias:

```powershell
npm install
```

Esto creará la carpeta `node_modules` y habilitará los comandos de Vite.

 
### 2. Ejecutar en modo desarrollo

Para levantar el servidor de desarrollo:

```powershell
npm run dev
```

El terminal mostrará la URL local donde se albergará la interfaz:

```
http://localhost:5173
```
